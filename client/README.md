# Фінансовий менеджер

Інформаційна система управління особистими фінансами.

## Можливості
- реєстрація і вхід користувача;
- додавання, редагування і видалення операцій;
- фільтрація за типом і датою;
- робота з бюджетами;
- аналітика і графік;
- модуль рекомендацій на основі історичних даних.
